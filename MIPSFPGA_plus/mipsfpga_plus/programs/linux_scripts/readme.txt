This folder contains the linux bash scripts. These files are the linux porting of the *.bat files.
For the usage it is necessary change the working directory in the related sub-program.
Usage example: 
cd 00_counter
sh ../linux_scripts/00_clean_all.sh

or
cd 03_pipeline_bypasses
sh ../linux_scripts/07_simulate_with_icarus.sh

*Release notes:
-Do you have installed and properly configured the MIPS compiler to run properly the scripts
-Do you have installed Modelsim to run properly the 06_simulate_with_modelsim.sh script
-Do you have installed icarus verilog to run properly the 07_simulate_with_icarus.sh script
